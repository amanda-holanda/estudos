# 1.2 Tipos núméricos (int, float, complex)
- Int: número inteiro. Ex: 1, 2 e 19.
- Float: número quebrado. Ex: 11.5 e 23.63.
- Complex: números imaginários. Ex: 10 + 1j.

[Operações]:
- +, -, *, **, /, //, %
- '/': divisão que retorna float
- '//': divisão que retorna inteiro
